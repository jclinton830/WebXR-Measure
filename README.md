# WebXR-Measure

Augmented Reality Measure with WebXR

## Compatibility

The WebXR Device API is not yet compatible with all browsers. Check https://caniuse.com/#feat=webxr for more information.

## Instructions

Navigate to [Demo](https://wonderful-albattani-95489f.netlify.app/) with your mobile device.

Follow these steps:
1. Press *START AR*.
2. Point your camera at a surface (wall, floor, table).
3. The perpendicular distance from your camera to the surface is shown live.

## Build

Install dependencies with

```
npm install
````

Start development server with

```
npm run serve
```

or build with

```
npm run build
```
